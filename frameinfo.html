FrameUnidraw

example application

FrameUnidraw is a library that extends ComUnidraw with multi-frame editing and display mechanisms.

overall framework objects

  • FrameEditor/FrameKit: editor with multi-frame extensions
  • FrameViewer: multi-frame viewer
  • FrameCreator/FrameCatalog: save/restore mechanism with multi-frame extensions

specialized graphic objects

  • FrameComp (FrameView/FrameScript): frame component and views
  • FramesComp (FramesView/FramesScript): multi-frame component and views
  • FrameIdrawComp (FrameIdrawView/FrameIdrawPS/FrameIdrawScript): multi-frame top-level component and views
  • FrameOverlaysComp (FrameOverlaysView/FrameOverlaysScript): OverlaysComp specialized for this framework

multi-frame editing commands

  • CreateFrameCmd/DeleteFrameCmd: frame creation/deletion commands
  • MoveFrameCmd/FrameBeginCmd/FrameEndCmd: frame nagivation commands
  • CreateMoveFrameCmd: macro command
  • FrameGroupCmd/FrameUngroupCmd/FrameFrontCmd/FrameBacktCmd: specialized structure commands
  • FrameCopyCmd: specialized copy command
  • FrameNewViewCmd: specialized new viewer command
