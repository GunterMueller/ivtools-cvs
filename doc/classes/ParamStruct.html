class ParamStruct (Return to index)

parameter definition structure for ParamList.

Type:
    Instantiable

Derived Classes

Include file:

    Attribute/paramlist.h

Description:

structure for defining an argument to be read out of a serialized view of an object written out for persistent storage or interprocess communication.

Public:

    ParamStruct (ParamStruct&)
      copy constructor.


    ParamStruct (const char* name, ParamFormat format, param_callback ifunc, int offset1, int offset2, int offset3, int offset4, int indirection = -1)
      construct with 'name' and 'format', a pointer to a static method with 'param_callback' signature, and up to 4 offsets relative to a base pointer that will be used in constructing the contents of another object during a de-serialization process (an istream constructor). Also possible to specify an 'indirection' amount used when adding offsets to the base.


    void* addr1 (void* base)
      compute address by adding offset1() to 'base' or an indirect base determined by the contents of 'base' + indirection().


    void* addr2 (void* base)
      compute address by adding offset2() to 'base' or an indirect base determined by the contents of 'base' + indirection().


    void* addr3 (void* base)
      compute address by adding offset3() to 'base' or an indirect base determined by the contents of 'base' + indirection().


    void* addr4 (void* base)
      compute address by adding offset4() to 'base' or an indirect base determined by the contents of 'base' + indirection().


    enum ParamFormat { required, optional, keyword, other }
      type of parameter formats supported. Similar to Common Lisp.


    ParamFormat format ()
      parameter format type: required, optional, keyword, or other.


    param_callback ifunc ()
      pointer to static method (or function) to invoke when parameter found during de-serialization process.


    int indirection ()
      offset from base address used to retrieve another address (by indirection). disabled if less than zero.


    const char* name ()
      the name of the parameter -- keyword parameters do not need the ":".


    int offset1 ()
      first offset from base. This one is always supplied, and might be the only true offset if pointing to an array.


    int offset2 ()
      second offset from base. Might really be an array length in a certain usage. The 'ifunc' is free to interpret these as it wants.


    int offset3 ()
      third offset from base.


    int offset4 ()
      fourth offset from base. If this is not enough either an array can can be passed by using the first two, or these can be ignored all together and you can use fixed locations within the object being deserialized.


    ~ParamStruct ()
