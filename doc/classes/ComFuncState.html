class ComFuncState (Return to index)

state object for holding invocation specific data about a ComFunc.

Type:
    Instantiable

Derived Classes

Include file:

    ComTerp/comfunc.h

Description:

object that holds the state of a ComFunc used in a particular context, which allows for nested and recursive use of a singular ComFunc.

Public:

    ComFuncState (const ComFuncState&)
      copy constructor.


    ComFuncState (int nargs, int nkeys, int pedepth=0, int command_symid=0)
      initialize with number of arguments (including arguments following keywords), number of keywords, an option post-eval depth (nesting within blocks of post-evaluation commands), and an optional symbol id associated with the ComFunc.


    int& command_symid ()
      symbol id associated with the ComFunc.


    int& nargs ()
      number of arguments (including arguments following keywords).


    int& nargskey ()
      number of arguments following keywords. Assumption of nargskey()<=nkey().


    int& nargspost ()
      number of unevaluated ComValue objects input to a post-evaluation ComFunc.


    int& nkeys ()
      number of keywords, ':' prefixed symbols in a list of arguments.


    int& npops ()
      number of pops since the execute method called.


    int& pedepth ()
      post-evaluation depth: the nesting within blocks of post-evaluation commands, within blocks of conditionally executing control commands.




Protected:

    int _command_symid


    int _nargs


    int _nargskey


    int _nargspost


    int _nkeys


    int _npops


    int _pedepth




Friend methods:

    friend ostream& operator << (ostream& s, const ComFuncState&)
      print contents to ostream, brief or not depending on associated ComTerp brief flag.
