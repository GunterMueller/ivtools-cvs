class Attribute (Return to index)

generic symbol/value pair.

Type:
    Instantiable

Derived Classes

Include file:

    Attribute/attribute.h

Description:

Attribute consists of a symbol, represented by its index into a symbol table, paired with a value, represented by an AttributeValue. Memory for the AttributeValue is owned by the Attribute.

Public:

    Attribute (const Attribute&)
      copy constructor.


    Attribute (const char* name =0, AttributeValue* value =0)
      construct an attribute by generating a symbol id for the 'name' character string, and accepting a pointer to an externally allocated 'value'.


    Attribute (int symid, AttributeValue* value =0)
      construct an attribute with a symbol id instead of a character string, and accepting a pointer to an externally allocated 'value'.


    char* Name ()
      look up and return symbol string.


    int SymbolId ()
      return the id of the symbol in the symbol table.


    AttributeValue* Value ()
      return a pointer to the internal AttributeValue.


    void Value (AttributeValue*)
      accept new pointer to an externally allocated AttributeValue, deleting the old pointer.


    virtual ~Attribute ()
      deallocate memory for internal AttributeValue.
